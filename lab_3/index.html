Розробка Інтернет клієнт-серверних систем:
Лабораторна робота №3

Варіант 1

1. Напишіть функцію isEmpty(obj), яка повертає true, якщо у об’єкта немає властивостей, в іншому випадку - false.

2. Створіть об’єкт calculator з трьома методами: - read() – запитує два значення за допомогою prompt і зберігає їх як властивості об’єкта. - sum() – повертає суму збережених значень. - mul() – множить збережені значення і повертає результат.

3. Створіть об’єкт week з днями тижня. Ключами в ньому будуть номери днів від початку тижня(понеділок – 1 і т.д.). Виведіть на екран поточний день тижня.

4. Напишіть функцію, яка глибоко знаходить перетин об’єктів і повертає об’єкт перетину. Наприклад: const data = { a: 1, b: { c: 3 } }; const data2 = { c: 1, b: { c: 3 } }; intersectionDeep(data, data2); // { b: { c: 3 } }

5. Напишіть функцію, яка робить глибоку перевірку на пустоту об’єкта. Пустими значеннями є: '', null, NaN, undefined, [], {} Наприклад: const data = { a: { b: undefined } }; const data2 = { a: { b: 1 } }; isEmptyDeep(data); // true isEmptyDeep(data2); // false

Лабораторна робота №9

Варіант 1

1. Напишіть функцію, яка викликає callback через певну затримку (наприклад, 2 секунди).

2. Напишіть функцію, яка фільтрує масив за допомогою callback-функції.

3. Напишіть функцію, яка виконує послідовність запитів до сервера з використанням callback для кожного запиту.

4. Напишіть функцію, яка використовує callback для керування потоками даних на клієнтській стороні.

5. Створіть Promise, який вирішується з певним результатом. Використайте метод .then() для обробки результату.

6. Використайте Promise.reject() для створення відхиленого Promise з певною помилкою.

7. Створіть Promise.race(), який залежить від найшвидшого завершення Promise з групи.

9. Створіть програму, яка виконує декілька паралельних асинхронних операцій з використанням async/await.

10.Створіть асинхронну функцію, яка використовує await для послідовного виконання операцій та обробки помилок з try/catch.

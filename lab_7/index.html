Лабораторна робота №7

Варіант 1

1. Створіть домішку EventMixin, яка містить методи on, off та trigger для роботи з подіями. Створіть клас Button та застосуйте до нього домішку EventMixin. Створіть екземпляр класу Button і використайте методи on, off та trigger для реєстрації, видалення та спровокування подій.

2. Створіть клас Vehicle як базовий клас для різних транспортних засобів. Створіть підкласи, наприклад, Car, Bicycle, Boat, які успадковують клас Vehicle. Створіть екземпляри кожного з класів транспортних засобів. Використайте оператор instanceof, щоб перевірити, до якого класу належить кожен екземпляр.

3. Розширте клас Number, додавши метод isPrime, який перевіряє, чи є число простим. Додайте метод factorial, який обчислює факторіал числа. Створіть змінну з числом і використайте нові методи для перевірки, чи є число простим, і обчислення його факторіалу.

4. Створіть клас BankAccount, у якого є приватний метод generateAccountNumber. Використайте WeakMap для збереження приватного методу generateAccountNumber. Напишіть публічний метод getAccountNumber, який використовує приватний метод generateAccountNumber для створення номера рахунку. Створіть екземпляр класу BankAccount та використайте публічний метод getAccountNumber для отримання номера рахунку.

5. Створіть клас StringUtils, у якому буде статичний метод reverseString, що приймає рядок і повертає його зворотнє значення. Додайте статичний метод countOccurrences, який приймає рядок та символ і повертає кількість входжень цього символу у рядок. Створіть екземпляр класу StringUtils і використайте статичні методи для обертання рядка та підрахунку кількості певних символів.
